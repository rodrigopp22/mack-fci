Esta pasta contém os códigos desenvolvidos durante a disciplina de Teoria dos Grafos.

```
O Projeto 1 consiste em implementar um algoritmo simples para decidir se um grafo é uma árvore ou não.
Para resolver este problema, basta considerar a quantidade de vértices e de arestasdo grafo e, também,
verificar se o grafo é conexo ou não. Apesar de existir um algoritmo eficiente e elegante para resolver
o problemada conexidade(e que será tratado posteriormente no curso), nesta tarefa implementaremos
um algoritmo mais básico. 

Os alunos deverão usar como base o arquivo Grafo.c, versão divulgada no início de março de 2021, e
completá-lo com funções para resolver o problema proposto: As estruturas de dados propostas devem
ser mantidas sem nenhuma alteração. O programa deverá ter uma função main que inicializa um grafo com
um valor fixo, verifica se ele é uma árvore e, depois, imprime os resultados obtidos.
```
